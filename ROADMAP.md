# Roadmap

Development roadmap for verenajs.

## Current Version: 2.0.0

---

## Completed (v2.0.0)

### Core Runtime
- [x] Event bus for component communication
- [x] Reactive state management (Store)
- [x] Theme system with dark/light modes
- [x] DOM utilities
- [x] Platform detection

### Component System
- [x] 1000+ UI components
- [x] 40+ component categories
- [x] CSS Modules styling
- [x] Factory pattern
- [x] Auto-discovery registry
- [x] Dynamic component loading
- [x] Component generation system

### Advanced Visual Builder
- [x] Professional drag-and-drop interface
- [x] Component palette with 1000+ components
- [x] Device preview (Desktop/Tablet/Mobile)
- [x] Property inspector with style editor
- [x] Layers tree for hierarchy
- [x] Canvas workspace with zoom
- [x] Undo/redo history (100 steps)
- [x] Code generation (verenajs, React, Vue, HTML)
- [x] Multi-page support
- [x] Keyboard shortcuts
- [x] Bottom panel (Console, Network, Performance, State)
- [x] Docker deployment dialog

### API Management
- [x] ApiClient with caching and retries
- [x] WebhookManager with HMAC signatures
- [x] BackendConnector (Node.js, Python, Go, PHP)
- [x] GraphQLClient
- [x] DataSourceManager
- [x] WebSocketManager

### Plugin System
- [x] Plugin lifecycle management
- [x] Hook system
- [x] Built-in plugins (analytics, auth, payments, email, charts, maps)
- [x] Plugin marketplace integration

### Docker Integration
- [x] Dockerfile generator
- [x] Docker Compose generator
- [x] Kubernetes manifest generator
- [x] CI/CD configuration (GitHub Actions, GitLab CI, Jenkins)

### Multi-Target Compiler
- [x] AST-based parser
- [x] Dependency analyzer
- [x] Tree shaking
- [x] Web bundle generator
- [x] Mobile generator (Capacitor)
- [x] Desktop generator (Electron/Qt)

### Platform Bridges
- [x] ZeroMQ communication layer
- [x] Qt native bridge
- [x] OpenCV integration

### Documentation
- [x] Comprehensive MD documentation
- [x] Interactive HTML documentation
- [x] Visual Builder guides
