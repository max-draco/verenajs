# Roadmap

Development roadmap for verenajs.

## Current Version: 2.0.0

---

## Completed (v2.0.0)

### Core Runtime
- [x] Event bus for component communication
- [x] Reactive state management (Store)
- [x] Theme system with dark/light modes
- [x] DOM utilities
- [x] Platform detection

### Component System
- [x] 257 UI components
- [x] CSS Modules styling
- [x] Factory pattern
- [x] Auto-discovery registry
- [x] Dynamic component loading

### Visual Builder
- [x] Drag-and-drop interface
- [x] Component palette
- [x] Property inspector
- [x] Canvas workspace
- [x] Undo/redo history
- [x] Code generation

### Multi-Target Compiler
- [x] AST-based parser
- [x] Dependency analyzer
- [x] Tree shaking
- [x] Web bundle generator
- [x] Mobile generator (Capacitor)
- [x] Desktop generator (Electron/Qt)

### Platform Bridges
- [x] ZeroMQ communication layer
- [x] Qt native bridge
- [x] OpenCV integration
